to run the project:

step 1 : clone the repo

step 2 : change the directory to localpdf using "cd localpdf"

step 3: run "npm install"

step 4: run "npm run dev"

step 5: change the directory to backend usnig "cd backend"

step 6: run "node server.js"

step 7: open the localhost provided in the terminal of the frontend and put the relative path of the folder from which you
want to extract the PDFs.
